# Retail Sales Analytics & Forecasting — Capstone Project

This project performs minimal end-to-end retail analytics, including:

- Data cleaning  
- Exploratory Data Analysis (EDA)  
- Baseline forecasting using Random Forest  
- 30-day iterative forecast  
- Store segmentation using KMeans  
- Minimal Streamlit dashboard skeleton  

## Files Included
- `capstone_minimal.ipynb` — Main notebook  
- `capstone_report.pdf` — Exported PDF report  
- `dashboard_app_minimal.py` — Streamlit dashboard skeleton  
- Forecast CSV & segmentation CSV  
- Processed data  
- PNG visualizations (EDA charts)

## Notes
- Original dataset is **not included** due to size/privacy.  
- Notebook is optimized for Google Colab + Google Drive.
